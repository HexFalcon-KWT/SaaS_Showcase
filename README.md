## ğŸ”§ Customization

All demos are fully customizable:
1. **Colors**: Modify theme colors in `globals.css`
2. **Components**: Edit components in `src/components/ui/`
3. **Content**: Update text, images, and data in demo pages
4. **Animations**: Adjust animation variants in `src/lib/animations.ts`
